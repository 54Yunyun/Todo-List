# Todo-List
Vue3 簡易版代辦清單

# Demo
https://todos-dusky-eight.vercel.app/

# 主要功能

- 可填入使用者名稱
- 添加待辦事項
- 添加待辦事項的類別 Business、Personal
- 添加 Check 已完成待辦事項
- 添加單筆刪除、多筆刪除
- 添加所有待辦事項筆數
